Creating a Minimal Landing Page

Overview

Create a minimal landing page for Dev Bootcamp.

Objectives

Create a Horizontal Strip

Create a horizontal strip of any color you like that says "Welcome to Dev Bootcamp" and occupies 100% of the browser window's width.

Ensure that scrollbars don't appear, even when you change the size of the browser window.

Add CSS Styles

Add some style: use text shadows and ensure they work in 2 major browsers. Use a CSS style attribute to give the strip a drop shadow appearance.
